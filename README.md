# Teu Gestor Backend

## Requisitos
- Node.js 18+
- Supabase Project (URL e Service Key)

## Instalação
```bash
npm install
```

## Ambiente
Copie o arquivo `.env.example` para `.env` e preencha com suas credenciais do Supabase.

## Desenvolvimento
```bash
npm run dev
```

## Build
```bash
npm run build
```

## Produção
```bash
npm start
```
